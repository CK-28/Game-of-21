Please follow the following steps in order to be able to play this game:

1. Download all files in the repo into one folder
2. Open a terminal window and navigate to the folder you created in step 1.
3. Type "ant compile" into your terminal window. The program should compile successfully here.
4. Type the command that appears when running step 3 into the terminal.
5. Enjoy the game!

*There is no need to recompile the program everytime you want to play the game. Step 4 is the only step needed.
